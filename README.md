# 🏏 Interactive Cricket Quiz

A beginner friendly command line Cricket Quiz developed in Python. This project allows users to test their cricket knowledge through multiple choice questions, choose different difficulty levels, calculate scores, and save results in a leaderboard using file handling.

## Features

- User name input
- Multiple difficulty levels
- Multiple choice cricket questions
- Automatic score calculation
- Leaderboard using text file
- Simple and beginner friendly interface
- File handling for storing player scores

## Technologies Used

- Python
- Jupyter Notebook
- Text File Handling

## How to Run

1. Download or clone this repository.
2. Open the notebook in Jupyter Notebook or Google Colab.
3. Run all the cells.
4. Enter your name.
5. Select a difficulty level.
6. Answer the quiz questions.
7. View your final score and leaderboard.
